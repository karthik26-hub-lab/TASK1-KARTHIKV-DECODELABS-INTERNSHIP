**Travique** is a premium, mobile-first web application designed to act as your minimalist digital travel concierge. Built with an absolute focus on cinematic aesthetics and frictionless user experience, it features gesture-driven interactions, a dynamic card-swiping interface, and a beautifully frosted UI.

---

## ✨ Core Features

*   **🃏 Native Flashcard Onboarding:** A buttery-smooth, native CSS `snap-x` scrolling carousel that introduces the app and collects your "Digital Passport" details before entry.
*   **🎙️ Voice AI & Swipe Deck:** A Tinder-style, swipeable card deck interface. Users can "speak" their travel vibe, and the app serves highly visual, curated destination matches. Swipe right to save to your Journal, swipe left to pass.
*   **📖 Cinematic Journal:** A personalized scrapbook where users can organize their saved destinations into folders (Beaches, Nature, etc.), add private notes, and track their trips.
*   **📍 Community Curator System:** Unlock the ability to publish your own "Hidden Gems" to the public feed complete with high-res imagery and direct **Google Maps URL** routing.
*   **📱 App-Like Navigation:** A floating, frosted-glass bottom dock with dynamic "pill" expansion animations for seamless tab switching without page reloads.
*   **🔔 Custom Toast Notifications:** Zero ugly browser alerts. All feedback is delivered via sleek, animated UI toast notifications.

## 🛠️ Tech Stack

Travique is built to be lightning-fast and entirely front-end, utilizing modern web standards without the overhead of heavy frameworks.

| Technology | Usage |
| :--- | :--- |
| **HTML5** | Semantic structure and accessibility. |
| **CSS3** | Custom animations, pure CSS scroll-snapping, and frosted glass (`backdrop-filter`) effects. |
| **JavaScript (Vanilla)** | DOM manipulation, drag/swipe physics for the card deck, and local storage state management. |
| **Tailwind CSS (CDN)** | Rapid utility-class styling and responsive layout constraints. |
| **FontAwesome** | Premium scalable vector icons. |
